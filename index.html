Flutter MVVM Base Project

(Model-View-ViewModel)

This is a base project for a any kind of Flutter application, following the MVVM architecture.

To get started, download the zip, extract it, and run the flutter pub get command. Then change the package name and app name according to your project requirements.

This base project includes enhancements and improvements such as handling common internet connection issues, easy-to-use local storage, common loading views, image loading with a shimmer effect, and more things comes in a easy way
